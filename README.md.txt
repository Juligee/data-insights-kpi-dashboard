PPEs Tracking Dashboard (Tableau Project)

This repository contains the Tableau workbook and supporting resources for the PPEs Tracking Dashboard, designed to monitor the availability, usage, and allocation of Personal Protective Equipment (PPE) across operations.

Project Structure
PPEs_Tracking/
│
├── data/                 
├── tableau/              
├── screenshots/          
└── README.md             

Project Objective

The goal of the PPEs Tracking Dashboard is to:

Track PPE consumption and availability

Identify shortages or supply risks

Monitor PPE distribution trends

Improve decision making for procurement and safety teams

Key Features

Total PPE stock and usage metrics

Department-level PPE usage analysis

Trend visualization over time

Interactive filters for quick insights

Export-ready dashboard for reporting

Tools & Technologies

Tableau Desktop / Tableau Public

Excel / CSV data sources

Version control with Git & GitHub

How to Use This Repository

Download the .twbx file from the tableau/ folder

Open it with Tableau Desktop or Tableau Public

Connect to the data files in the data/ directory if needed

Explore and interact with the dashboard
