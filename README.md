Overview

This project explores global consumer spending patterns across multiple economic
regions. Using Python, and Excel, the analysis
highlights spending trends and provides business‑relevant insights.

Key Objectives

1.  Identify major spending categories globally.
2.  Compare region‑to‑region spending behaviors.
3.  Visualize changes over time using charts and dashboards.
4.  Provide insights relevant for business strategy, product
    positioning, and market expansion.

Files Included

-   project_report_template.pdf – PDF report with visuals
-   dataset_sample.csv – Sample dataset used for analysis, with Dashboards
-   project1_analysis_notebook - Jupyter notebook for Data Wrangling and Analysis

Tools Used

-   Python (Pandas, Matplotlib)
-   Excel
-   Jupyter Notebook

Business Insights Highlight

-   Essential goods spending remains stable globally within the captured period, while
    discretionary categories are more sensitive to economic shifts.
-   Emerging markets show rapid growth in digital services spending.
-   Countries with higher GDP maintain more stable spending patterns
    over time.
